## BuddiesGram, a Social Network

### Description
> ## BuddiesGram, a Social Network built for educational purpose. The purpose was to learn Flutter by cloning Instagram. Most of Instagram Functionalities are integrated in this project. Only the messaging hasn't been added.

## What I used
- Flutter
- Firebase
